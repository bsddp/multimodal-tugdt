# Data handling

Only generated synthetic examples belong in version control.

- `synthetic/`: public software fixtures with no real participants
- `raw/`: original research files; ignored by Git
- `interim/`: intermediate transformations; ignored by Git
- `processed/`: derived time series and features; ignored by Git

Do not add names, faces, natural voices, hospital identifiers, identifiable clinical records,
or data whose consent and governance do not explicitly permit public release.
